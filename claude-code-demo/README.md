# Learn AI — A Friendly Introduction

A simple, single-page educational website that explains artificial intelligence in plain language for a non-technical audience. No jargon, no equations — just clear explanations of what AI is, how it works, and where it shows up in everyday life.

## Contents

- **Hero** — introduction and call to action
- **What is AI?** — a plain-language overview
- **How it works** — a simple 3-step explanation
- **Examples** — everyday tools that already use AI
- **FAQ** — answers to common questions

## Usage

This is a single self-contained HTML file with embedded CSS — no build tools, dependencies, or setup required.

1. Download `index.html`
2. Open it in any web browser

## Customizing

- Colors and fonts are defined as CSS variables near the top of the `<style>` block (`--bg`, `--ink`, `--accent`, etc.)
- All text content is placeholder copy and can be edited directly in the HTML
- The layout is responsive and adjusts for mobile screens below 680px width
